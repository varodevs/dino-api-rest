Welcome to the Dino API

This is the homepage of the Dino API.

Use the following endpoints to access the API:

  • GET /dinos - Get all dinosaurs
  • GET /dinos/random - Get a random dinosaur
  • GET /dinos/name/sorted - Get all dinosaurs alphabetically sorted
  • GET /dinos/:id - Get a specific dinosaur by id
  • POST /dinos/name/:name - Get a specific dinosaur by name
